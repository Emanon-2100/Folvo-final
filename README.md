# Folvo - 您的智能记账伙伴（PKU程序设计实习大作业）

Folvo 是一款使用 Qt 开发的桌面智能记账本。它通过一个新颖直观的界面，帮助用户轻松记录和管理日常收支，并为未来的财务规划提供支持。

---

## ✨ 主要功能

- **快速记账**: 简洁的弹窗，快速录入每笔交易。
- **直观界面**: 独特的图片主体布局，结合功能按钮栏，操作一目了然。
- **账单流水**: 在独立的子窗口中，以清晰的表格形式查看所有历史账目。
- **数据持久化**: 支持将账目数据保存到本地 JSON 文件，并能随时加载。
- **跨平台**: 基于 Qt 构建，可在 Windows, macOS, Linux 上运行。
- **未来规划**: 还有统计报表、预算计划、AI 财务建议等功能。
